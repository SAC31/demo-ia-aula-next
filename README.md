# Demo IA para Aula (Next.js)

Aplicação interativa para **executive education** focada em direção comercial.

## O que demonstra

- Trilha iterativa em 4 etapas (fundamentos, prompts, chatbot e assistente virtual)
- Comparador de prompts (fraco, médio e forte)
- Simulador de cenário comercial (prospecção e objeção de preço)
- Laboratório de prompt com pontuação e feedback em tempo real

## Como executar

```bash
npm install
npm run dev
```

Abra `http://localhost:3000`.

## Build

```bash
npm run build
```
